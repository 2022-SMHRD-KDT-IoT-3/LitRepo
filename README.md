## 소개
팀명 : LIT! ✨
제품명 : Dream Catcher
- 심박센서를 이용한 수면데이터 수집 무드등 + 피드백 서비스
- Challenge Point : 헬스케어

## 주요기능
- 심박센서, 데시벨 센서, 온/습도 센서로 수면 데이터 수집
- 무드등 색상 출력
- ASMR 재생
- 회원관리 기능(로그인, 회원가입, 회원정보수정)
- 커뮤니티 게시판
- 웹을 통해 수면 데이터 확인 후 맞춤 피드백 제공
- 콘텐츠 추천

## 개발환경
- 사용언어 : Java 15 / HTML / CSS / Javascript
- 개발 도구 : eclipse / VSCode
- 서버 환경 : tomcat v9.0
- 데이터베이스 : oracle 11g
- 디자인도구 : PhotoShop
